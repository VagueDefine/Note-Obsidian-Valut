<font style="color:rgba(0, 0, 0, 0.85) !important;">达林顿管（Darlington Transistor）又称复合管，是由两个或多个三极管组合而成的一种晶体管结构，其原理如下：</font>

### <font style="color:rgb(0, 0, 0);">结构组成</font>
+ <font style="color:rgba(0, 0, 0, 0.85) !important;">达林顿管通常由两个三极管组成，这两个三极管的连接方式有多种，常见的有 NPN 型和 PNP 型两种组合方式。以 NPN 型达林顿管为例，第一个三极管的集电极与第二个三极管的基极相连，第一个三极管的发射极直接作为达林顿管的发射极，第二个三极管的集电极作为达林顿管的集电极，而第一个三极管的基极则作为达林顿管的基极。</font>

### <font style="color:rgb(0, 0, 0);">工作原理</font>
+ **<font style="color:rgb(0, 0, 0) !important;">电流放大</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：达林顿管的主要作用是实现较大的电流放大倍数。当有一个较小的基极电流</font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251059.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">流入第一个三极管时，根据三极管的电流放大作用，会在第一个三极管的集电极产生一个较大的集电极电流</font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251058.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">，其中</font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251060.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">是第一个三极管的电流放大系数。这个</font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251058%201.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">同时作为第二个三极管的基极电流</font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251060%201.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">，经过第二个三极管再次放大，在第二个三极管的集电极产生更大的电流</font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251059%201.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">。所以，达林顿管总的电流放大倍数</font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251061%201.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">，可以达到几百甚至上千，能够将微弱的信号电流放大到足够大的程度，以驱动负载。</font>
+ **<font style="color:rgb(0, 0, 0) !important;">电压放大</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：在一些电路中，达林顿管也能实现一定的电压放大作用。由于其电流放大能力很强，当它连接在合适的电路中时，通过负载电阻的作用，可以将放大后的电流变化转化为电压变化，从而实现对输入信号的电压放大。不过，达林顿管的电压放大倍数通常不如专门的电压放大电路高，但其突出优势在于电流放大能力。</font>

### <font style="color:rgb(0, 0, 0);">特点与应用</font>
+ **<font style="color:rgb(0, 0, 0) !important;">特点</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：</font><font style="color:#DF2A3F;">达林顿管具有高电流放大倍数、高输入阻抗、低输出阻抗等特点。但也存在一些缺点，如饱和压降较高、开关速度相对较慢等。</font>
+ **<font style="color:rgb(0, 0, 0) !important;">应用</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：常用于需要较大电流驱动能力的电路中，如音频功率放大器、电机驱动电路、继电器驱动电路等。在音频功率放大器中，达林顿管可以将前置放大电路输出的微弱音频信号电流放大到足够大，以驱动扬声器发出声音；在电机驱动电路中，能够提供足够的电流来控制电机的转动。</font>

### <font style="color:rgb(0, 0, 0);">电气参数</font>
+ **<font style="color:rgb(0, 0, 0) !important;">集电极 - 发射极电压（</font>**![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251060%202.svg)**<font style="color:rgb(0, 0, 0) !important;">）</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：应大于电路中达林顿管集电极与发射极之间可能出现的最大电压，以确保管子不会因承受过高电压而被击穿。例如，在一些开关电源电路中，当开关管截止时，</font><font style="color:#DF2A3F;">集电极会出现较高的反峰电压（对于感性器件需要特别注意）</font><font style="color:rgba(0, 0, 0, 0.85) !important;">，此时就需要选择</font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251060%202.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">足够高的达林顿管来承受这种电压冲击。也可以设计感性负载防护电路。</font>[硬件设计——感性负载防护_感性负载保护电路-CSDN博客](https://blog.csdn.net/m0_38112603/article/details/124252953#:~:text=%E6%84%9F%E6%80%A7%E8%B4%9F%E8%BD%BD%E9%A9%B1%E5%8A%A8%E7%94%B5%E8%B7%AF%E5%9C%A8%E5%85%B3%E6%96%AD%E6%97%B6%E7%94%B5%E6%84%9F%E4%B8%AD%E6%89%80%E5%AD%98%E5%82%A8%E7%9A%84%E8%83%BD%E9%87%8F%E4%BC%9A%E7%9E%AC%E9%97%B4%E9%87%8A%E6%94%BE%EF%BC%88E%20%3D%201%2F2LII%20%E2%89%88,Pt%20%3D%20UIt%EF%BC%89%EF%BC%8C%E4%BA%A7%E7%94%9F%E5%B0%96%E5%B3%B0%E7%94%B5%E5%8E%8B%E5%92%8C%E8%84%89%E5%86%B2%E6%8A%96%E5%8A%A8%EF%BC%88EMF%E3%80%81EMI%EF%BC%89%E3%80%82%20%E5%A6%82%E6%9E%9C%E6%97%A0%E9%98%B2%E6%8A%A4%EF%BC%8C%E5%88%99%E4%BC%9A%E7%9E%AC%E9%97%B4%E9%87%8A%E6%94%BE%E7%94%B5%E5%8E%8B%EF%BC%88%E2%89%88%E4%BE%9B%E7%94%B5%E7%94%B5%E5%8E%8B%E7%9A%8410%E5%80%8D%EF%BC%89%EF%BC%8C%E7%9E%AC%E9%97%B4%E9%87%8A%E6%94%BE%E7%94%B5%E5%8E%8B%E4%B8%8E%E7%94%B5%E6%84%9F%E9%87%8F%E6%9C%89%E5%85%B3%EF%BC%8C%E7%94%B5%E6%84%9F%E9%87%8F%E8%B6%8A%E5%A4%A7%E7%9E%AC%E9%97%B4%E7%94%B5%E5%8E%8B%E8%B6%8A%E5%A4%A7%EF%BC%8C%E8%84%89%E5%86%B2%E6%8A%96%E5%8A%A8%E6%97%B6%E9%97%B4%E8%B6%8A%E9%95%BF%E3%80%82)
+ **<font style="color:rgb(0, 0, 0) !important;">集电极电流（</font>**![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251061%203.svg)**<font style="color:rgb(0, 0, 0) !important;">）</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：要根据负载所需的电流来选择。达林顿管的最大集电极电流应大于负载的最大工作电流，并且需要考虑一定的余量，以应对可能出现的电流峰值。比如在驱动小型直流电机时，要根据电机的额定电流以及启动时的冲击电流来选择合适</font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251061%203.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">的达林顿管。</font>
+ **<font style="color:rgb(0, 0, 0) !important;">电流放大系数（β）</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：根据具体电路对信号放大倍数的要求来选择。如果需要将微弱的信号进行较大倍数的放大，就需要选择 β 值较高的达林顿管。但需要注意的是，β 值过高可能会导致管子的稳定性下降，所以要在满足放大要求的前提下，尽量选择 β 值适中的管子。</font>
+ **<font style="color:rgb(0, 0, 0) !important;">集电极 - 基极反向饱和电流（</font>**![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251061.svg)**<font style="color:rgb(0, 0, 0) !important;">）</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：</font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251061.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">越小，说明管子的反向漏电越小，性能越好。在一些对漏电要求较高的电路中，如高精度的传感器电路，应选择 </font>![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251061.svg)<font style="color:rgba(0, 0, 0, 0.85) !important;">小的达林顿管，以减少漏电对电路性能的影响。</font>

### <font style="color:rgb(0, 0, 0);">开关特性</font>
+ **<font style="color:rgb(0, 0, 0) !important;">开关速度</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：对于需要频繁开关的电路，如数字逻辑电路、PWM 调制电路等，要选择开关速度快的达林顿管，以减少开关过程中的能量损耗和信号失真。一般来说，高频特性好的达林顿管开关速度也较快，可以通过查看数据手册中的开关时间参数（如上升时间、下降时间、存储时间等）来评估。</font>
+ **<font style="color:rgb(0, 0, 0) !important;">反向恢复时间</font>**![image](0000assets/4、达林顿管（Darlington%20Transistor）/file-20251224160251061%202.svg)**<font style="color:rgb(0, 0, 0) !important;">（</font>**<font style="color:rgba(0, 0, 0, 0.85);">Reverse Recovery Time</font>**<font style="color:rgb(0, 0, 0) !important;">）</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">：在一些包含感性负载的电路中，当达林顿管关断时，感性负载会产生反向电动势，导致管子承受反向电压。此时，反向恢复时间短的达林顿管能够更快地恢复到截止状态，减少反向电流的持续时间，从而降低管子的功耗和损坏风险。</font>
